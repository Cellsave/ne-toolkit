About WHOIS

            
WHOIS is a query and response protocol that is widely used for querying databases that store the registered users or assignees of domain names and IP address blocks:

            
    
                
  • Domain registration information
    • 
                
  • Registrar details
    • 
                
  • Creation and expiration dates
    • 
                
  • Name servers
    • 
                
  • Contact information (when not protected)
    • 
                
  • IP address ownership and allocation details
    • 
            

            
            
This implementation uses public WHOIS services and IP geolocation databases for accurate and comprehensive lookups.
