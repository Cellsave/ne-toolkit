Getting Started

Welcome to the Network Engineer's Interactive Toolbox! This suite of tools is designed to help network engineers with common tasks and troubleshooting.

Available Tools

  • Troubleshooting Tools - Generate vendor-specific commands for network troubleshooting
  • Configuration Convertor - Convert between different JUNOS configuration formats
  • IP Subnet Calculator - Calculate subnet information and IP ranges
  • Password Decrypt - Decrypt common network device password formats
  • WHOIS Lookup - Look up domain and IP ownership information
  • BGP Analysis Tools - Analyze BGP routes and AS information

Navigation

Use the navigation bar at the top of the page to access different tools. Each tool has its own dedicated page with specific functionality.

Tool-Specific Help

Troubleshooting Tools

This tool helps generate vendor-specific troubleshooting commands for network issues.

  1. Select the vendor (Cisco or Juniper)
  2. Choose the software version
  3. Select the protocol or feature you're troubleshooting
  4. Optionally paste device configuration for analysis
  5. Describe the issue you're experiencing
  6. Click "Generate Commands" to get troubleshooting commands

Configuration Convertor

Convert between different JUNOS configuration formats:

  1. Paste your JUNOS configuration in the input box
  2. Select the conversion type (set to hierarchical, hierarchical to set, etc.)
  3. Click "Convert Configuration" to process
  4. Copy the converted output for use in your devices

IP Subnet Calculator

Calculate subnet information for IP networks:

  1. Enter an IP address (e.g., 192.168.1.1)
  2. Enter a subnet mask (e.g., 255.255.255.0) or CIDR notation (e.g., /24)
  3. Click "Calculate Subnet" to see detailed subnet information

Password Decrypt

Decrypt common network device password formats:

  1. Enter the encrypted password string
  2. Select the vendor/encryption type
  3. Click "Decrypt Password" to see the decrypted result

WHOIS Lookup

API Provider: WhoisXML API

Look up domain and IP ownership information:

  1. Enter a domain name or IP address
  2. Select the lookup type (WHOIS or IP Geolocation)
  3. Click "Perform Lookup" to retrieve information

BGP Analysis Tools

Looking Glass Provider: BGP.Tools

Analyze BGP routes and autonomous system information:

  1. Enter a prefix or IP address
  2. Select the analysis tool
  3. Click "Analyze" to get BGP information
